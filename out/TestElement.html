Class: TestElement

TestElement(key, auxiliary)

A class that creates an element for testing. Each linked list node will have an object that is an instance of this class as its value. This class will have two instance variables. First is the number which we will be sorting against and the second is an auxiliary value which is the original position in the linked list.
