Class: LinkedListNode

LinkedListNode(data, next)

The LinkedListNode class represents each node in a linked list. Each node contains an object of the Test Element class.

Constructor

new LinkedListNode(data, next)

Constructor for creating a new LinkedListNode. It accepts the data that will be stored in this node and the next node.
Parameters:
Name Type Description
data TestElement The test element object is the data that is associated with a node.
next LinkedListNode The next element after this node.
